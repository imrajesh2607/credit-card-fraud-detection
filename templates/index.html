Credit Card Fraud Detection

The Objective of this project is to predict whether a credit card transaction is fraudulent or not, based on the transaction time, amount and other transaction related data. It track down the fraud credit card transaction, which is done by detecting anomalies in the transaction data using the Machine Learning Algorithm of Logistic Regression.
